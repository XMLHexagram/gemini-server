# Gemini-server

An easy-to-use Gemini Server developed under [go-conflict](https://github.com/go-conflict/conflict)

## Overview

Gemini-server是一个实现了Gemini协议并且易于使用的Gemini Server，致力于只需要简单的修改配置文件，就可以开始使用。
